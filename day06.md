# Git Day 6

## What is a Fork?

A fork is a personal copy of another repository.

## What is a Pull Request?

A pull request is a request to merge changes into another branch.

## What is an Issue?

Issues are used to track bugs, tasks, and feature requests.

## What is Code Review?

Code review is the process of reviewing changes before merging.

## Commands Learned

git push origin branch-name

git branch -d branch-name

git push origin --delete branch-name

## Pull Request Workflow

Create Branch
↓
Make Changes
↓
Commit
↓
Push
↓
Pull Request
↓
Review
↓
Merge

## Key Takeaway

Pull Requests are the standard way teams collaborate using GitHub.
