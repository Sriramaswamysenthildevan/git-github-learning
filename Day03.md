# Git Day 3

## What is a Branch?

A branch is an independent line of development that allows developers to work on features without affecting the main codebase.

## Commands Learned

git branch
git switch
git switch -c
git merge
git branch -d

## Branch Workflow

main
│
└── feature-branch

feature-branch
↓
commit
↓
merge
↓
main

## Key Takeaways

- Branches isolate work.
- Multiple developers can work simultaneously.
- Merging combines changes.
