# Git Day 5

## What is git restore?

Restores files to their last committed state.

## What is git restore --staged?

Removes files from the staging area.

## What is git reset?

Moves HEAD to a previous commit.

### Types

git reset --soft
git reset --mixed
git reset --hard

## What is git revert?

Creates a new commit that reverses a previous commit.

## Difference

restore = undo file changes

reset = move history

revert = safely undo commits

## Commands Learned

git restore
git restore --staged
git reset
git revert

## Key Takeaway

Use revert for shared repositories.
Use reset carefully.
